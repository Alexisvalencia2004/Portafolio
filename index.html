Proyectos

Capchat

La página tiene un diseño limpio y funcional, centrado en un ejercicio visual que reta al usuario a seleccionar todas las imágenes que representan un carro entre un conjunto de opciones. Esto podría usarse como un juego educativo, un sistema de validación visual o incluso como parte de un proceso de autenticación.

Calculadora Básica

Esta página web presenta una calculadora interactiva básica, diseñada para realizar operaciones matemáticas simples como suma, resta, multiplicación y división. Su diseño es limpio, intuitivo y funcional, ideal para usuarios que necesitan realizar cálculos rápidos de manera sencilla.

Calculadora de Índice de Masa Corporal

Esta página web es una herramienta interactiva que permite a los usuarios calcular su Índice de Masa Corporal (IMC) ingresando sus datos personales, como peso y estatura. Es ideal para quienes desean conocer su estado de salud relacionado con el peso corporal de forma rápida y sencilla.
